# Drip Store - Front-end

Este é o projeto final do módulo de Front-end do meu curso de Desenvolvimento Fullstack pelo Geração Tech 3.0. A aplicação é um e-commerce moderno e responsivo, focado na experiência de compra de calçados e acessórios.

**🔗 [Confira o Deploy do Projeto Aqui](https://projeto-front-end-loja-drip.vercel.app/)**

---

### 🚀 Sobre o Projeto

O objetivo principal foi construir uma interface funcional utilizando **React**, aplicando conceitos de roteamento dinâmico, gerenciamento de estado global e estilização utilitária. O projeto simula um fluxo real de navegação, desde a vitrine de produtos até a persistência do carrinho de compras.

### ✨ Funcionalidades Principais

- **Shopping Experience:** Navegação fluida com carrosséis promocionais automáticos e destaques de produtos.
- **Filtros Inteligentes:** Sistema funcional de filtragem por categoria.
- **Páginas de Detalhes:** Visualização individual de produtos com seleção de imagens, tamanhos e cores.
- **Gestão de Carrinho:** Adição de produtos com persistência de dados via `localStorage` (os itens permanecem salvos mesmo após atualizar a página).
- **Design Responsivo:** Layout _mobile-first_ baseado em design do Figma, totalmente adaptado para dispositivos móveis e desktop.

### 🛠 Tecnologias Utilizadas

O projeto foi construído com as seguintes ferramentas:

- ⚛️ **React (com Vite):** Para uma estrutura de desenvolvimento rápida e moderna.
- 📦 **React Router DOM:** Gerenciamento de rotas e navegação entre páginas.
- 🎨 **TailwindCSS:** Estilização eficiente, responsiva e moderna.
- 🗂 **Context API:** Para gerenciar o estado global do carrinho de compras.
- 🧠 **JavaScript (ES6+):** Lógica funcional de filtros e manipulação de dados.
- 💾 **localStorage:** Para manter os itens do carrinho salvos no navegador.

### ⚙️ Como executar o projeto localmente

Se desejar testar o projeto na sua máquina, siga os passos abaixo:

1.  **Clone o repositório:**

    ```bash
    git clone https://github.com/paulo-alencar/projeto-front-end-loja-drip
    ```

2.  **Instale as dependências:**

    ```bash
    npm install
    ```

3.  **Inicie o servidor de desenvolvimento:**

    ```bash
    npm run dev
    ```

4.  **Acesse no navegador:**
    Abra `http://localhost:5173` para visualizar a aplicação.

### 🧠 Competências Aplicadas

Neste projeto, foquei na aplicação prática de padrões de mercado utilizando o ecossistema React. O desenvolvimento priorizou a organização do código e o fluxo de dados entre componentes:

- **Arquitetura de Componentes:** Estruturação da interface em partes reutilizáveis e organizadas.
- **Estado Global (Context API):** Implementação de contexto para gerenciamento do carrinho em toda a aplicação.
- **Roteamento Dinâmico:** Uso de `React Router DOM` para navegação entre páginas (SPA).
- **Estilização Utilitária:** Aplicação de **TailwindCSS** para um layout responsivo e _mobile-first_.
- **Persistência de Dados:** Uso de `localStorage` para manter os itens do carrinho salvos no navegador.
